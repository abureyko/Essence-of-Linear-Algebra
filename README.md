# Essence of Linear Algebra

Учебно-исследовательский проект по линейной алгебре на Python: от геометрии векторов до собственных векторов, смены базиса, определителей и линейных преобразований.

Проект сделан как визуальный конспект: каждая тема сопровождается небольшим исполняемым примером, где абстрактная формула превращается в график, эксперимент или проверку вычислений.

## Идея проекта

Линейная алгебра часто выглядит как набор правил для матриц, но за ними стоит геометрия: растяжения, повороты, проекции, сжатия пространства, смена координат и преобразование базисов.

Цель проекта - показать эти идеи через код:

- векторы рассматриваются как направленные отрезки и как координаты;
- матрицы интерпретируются как преобразования пространства;
- определитель связывается с изменением площади;
- ранг, ядро и пространство столбцов объясняются визуально;
- скалярное произведение показывается через проекцию и линейное преобразование;
- собственные векторы рассматриваются как направления, которые матрица не поворачивает.

## Почему это можно представить как достижение

Этот репозиторий можно оформить как учебно-методическую публикацию или проектную работу, потому что он объединяет:

- программную реализацию базовых понятий линейной алгебры;
- визуальные эксперименты для объяснения геометрического смысла;
- последовательную структуру тем от простых векторов к собственным векторам;
- воспроизводимые Python-скрипты, которые можно запускать и модифицировать;
- русскоязычные комментарии, полезные для студентов, изучающих тему впервые.

Проект может быть представлен как открытый образовательный материал по теме:  
**"Визуализация фундаментальных понятий линейной алгебры средствами Python"**.

## Быстрый запуск

Клонируйте проект и установите зависимости:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Запустите любой модуль:

```bash
python 03_linear_transformations.py
python 06_matrix_determinant.py
python 09_dot_product.py
```

Для раздела с линейной оболочкой:

```bash
cd 02_linear_span
python visualize_span.py
```

## Как обновить графики

Все изображения из раздела с визуализациями создаются самими уроками и сохраняются в `assets/plots/`.

Если нужно пересоздать галерею, можно запустить уроки по одному:

```bash
python 01_vector_operations.py
python 03_linear_transformations.py
python 06_matrix_determinant.py
python 14_eigen_vectors.py
```

На сервере или в среде без графического окна удобно использовать backend `Agg`:

```bash
MPLBACKEND=Agg python 09_dot_product.py
```

## Пример: линейная комбинация

```python
from linear_combinations import linear_combination

i_hat = [1, 0]
j_hat = [0, 1]

result = linear_combination([i_hat, j_hat], [3, 2])
print(result)  # [3, 2]
```

Математически это означает:

```text
3 * i + 2 * j = [3, 2]
```

То есть точка `[3, 2]` получается как сумма трех шагов по оси `x` и двух шагов по оси `y`.

## Методическая ценность

Проект построен вокруг принципа: сначала увидеть, потом формализовать.

Вместо того чтобы начинать с сухого определения матрицы, скрипты показывают, как матрица действует на пространство. Например:

- матрица поворота меняет направления базисных векторов;
- матрица с нулевым определителем "схлопывает" площадь;
- произведение матриц соответствует последовательному применению преобразований;
- смена базиса меняет координаты вектора, но не сам геометрический объект.

Такой подход помогает студенту связать вычисления с интуицией.
